## CRUD Laravel Data Batu Permata

### Deskripsi

Web ini merupakan sistem CRUD sederhana untuk mengelola data batu permata menggunakan Laravel dan MySQL.

### Fitur

* Tambah data batu permata
* Menampilkan data batu permata
* Mengedit data batu permata
* Menghapus data batu permata

### Teknologi

* Laravel
* PHP
* MySQL

### Cara Instalasi

Clone repository:

```
git clone https://github.com/username/crudbatupermata.git
```

Masuk ke folder:

```
cd crudbatupermata
```

Install dependency:

```
composer install
```

Jalankan server:

```
php artisan serve
```

Buka di browser:

```
http://127.0.0.1:8000
```
